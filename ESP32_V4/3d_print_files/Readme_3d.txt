
3d files are alpha versions.

layout_4_inch is a template for the specified project box. It should be laid on top of the project box lid and used to mark the cut-out for the TFT screen & mounting holes and centre hole markings for the encoder & button locations.


bezel_4_inch is to print a bezel which fits around the TFT screen, covering the gap between the cut-out and TFT screen.


Please note, these files have been drawn but not printed.

If you print them, please give feedback as to if they are ok, or any errors that need correcting
